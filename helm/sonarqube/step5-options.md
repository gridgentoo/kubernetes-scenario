SonarQube не ограничивается только Java. Он предлагает **code analysis** этих 27+ языков:

![Multiple languages](/javajon/courses/kubernetes-pipelines/sonarqube/assets/multiple-languages.png "Code Analysis for multiple languages.")

ABAP · Apex · C/C++ · C# · CSS · COBOL · Flex · Go · HTML · Java · JavaScript · Kotlin · Objective-C · PL/SQL · PL/I · PHP · Python · RPG · Ruby · Scala · Swift · T-SQL · TypeScript · VB.NET · VB6 · XML

Больше языков и возможностей обсуждается в [community forum](https://community.sonarsource.com/) и [roadmap](https://www.sonarqube.org/roadmap/).

Наряду с растущей поддержкой языков существует [богатая экосистема из 70+ плагинов](https://docs.sonarqube.org/display/PLUG/Plugin+Library) управляемая сообществом **SonarSource.**

**SonarQube** и его **community of plugins** учат всех нас [стать лучшими инженерами](https://en.wikipedia.org/wiki/List_of_software_bugs). Подумайте об участии в этих проектах с открытым исходным кодом или [присоединитесь к команде **SonarSource**](https://www.sonarsource.com/company/jobs/).
